## Introduction

Calliope is a work in progress Lavalink V4 library for Ruby.

## Installation

Currently, you'll have to build the gem yourself. But, in the future, Calliope will be available as an installable gem.

1. **Install the latest version of Ruby**

This should be fairly self-explanatory. See [here](https://www.ruby-lang.org/en/documentation/installation/) for instructions.

2. **Usage**

Either add `gem calliope` to your Gemfile, or do `$ gem install calliope`.
